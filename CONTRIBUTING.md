# Contributing

Contributions must be original or use clearly identified compatible material. Record all external
dependencies, assets, generated inputs, and license obligations in `THIRD_PARTY_NOTICES.md`.

For substantial changes, include:

- capability and protocol impact;
- adapter/version compatibility impact;
- tests or fixtures proving observable behavior;
- security and side-effect classification;
- provenance declaration for external material.

Do not add generated build outputs, caches, logs, local worlds, credentials, or game assets.
